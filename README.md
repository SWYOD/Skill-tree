# Skill Tree

Персональный менеджер навыков в виде игрового дерева скиллов (в духе Watch Dogs):
радиальный анимированный граф в центре, панель-дерево слева и инспектор/чеклист/заметка справа.

## Возможности (MVP)

- Главные ветки, под-ветки и узлы навыков (радиальная раскладка вокруг центра).
- Неоновое свечение, «заряжающиеся» коннекторы, pop-анимация и разблокировка узлов.
- Чеклисты внутри узла с автоматическим расчётом прогресса в %.
- Markdown-заметка (`.md`) на каждый узел.
- Выбор цвета ветки и отдельного цвета/иконки узла.
- Импорт/экспорт всего дерева или отдельной ветки в JSON, экспорт графа в PNG.
- Панель «последний изменённый», сводная статистика в тулбаре.

## Хранение

Данные лежат в выбранной вами директории:

```
<корень>/
  store.json        # структура дерева, чеклисты, цвета, иконки
  notes/<id>.md     # заметки узлов (можно положить директорию внутрь Obsidian-vault)
```

## Запуск

```bash
npm install
npm run dev      # запуск в режиме разработки (Electron)
```

Прочие команды:

```bash
npm run build      # production-сборка в out/
npm run typecheck  # проверка типов
```

## Стек

Electron · React · TypeScript · Vite (electron-vite) · Zustand · d3-hierarchy ·
Framer Motion · lucide-react.
